## Build and run the server

```bash
git clone https://github.com/ringostarr80/rust-vms2-tile-server.git
cd rust-vms2-tile-server
cp .env.example .env
# modify the .env file!
cargo run # this starts the HTTP Server. Open your browser and go to http://127.0.0.1:8000
```
